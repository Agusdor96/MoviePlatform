Acerca de AguMovie

AguMovie es un proyecto creado con pasión y dedicación, destinado a los amantes del cine que desean una plataforma sencilla y eficiente para gestionar su colección de películas. Este proyecto no solo se enfoca en la funcionalidad, sino también en brindar una experiencia agradable para los usuarios finales.

Objetivos del Proyecto

El principal objetivo de AguMovie es proporcionar una interfaz intuitiva para gestionar películas, permitiendo a los usuarios agregar sus películas favoritas fácilmente. La idea es combinar tecnologías modernas para ofrecer un rendimiento óptimo y una experiencia de usuario fluida.

Tecnologías Utilizadas

  • Backend: Utilizamos Node.js con el framework Express.js para construir una API robusta que maneja las operaciones CRUD.
  • Base de Datos: MongoDB, junto con Mongoose como ODM, nos permite almacenar y gestionar los datos de las películas de manera eficiente.
  • Frontend: HTML, CSS y JavaScript son la base de nuestra interfaz de usuario, con Axios para manejar las solicitudes HTTP entre el cliente y el servidor.

Problemas Abordados y Soluciones

Sincronización de Datos entre Frontend y Backend:

Problema: Inicialmente, al agregar o eliminar películas desde el frontend, los cambios no se reflejaban automáticamente en la interfaz de usuario.
Solución: Implementamos llamadas asíncronas con Axios y manejamos la actualización del DOM de manera eficiente para asegurar que cualquier cambio en la base de datos se refleje inmediatamente en la interfaz de usuario.

Manejo de Formularios y DOM

Problema: Manipulaciones incorrectas del DOM causaban errores cuando ciertos elementos no estaban presentes.
Solución: Implementamos verificaciones para asegurar que los elementos existen antes de intentar manipularlos, y añadimos funcionalidad para resetear el formulario tras su envío.

Reflexiones Finales

AguMovie es más que un simple proyecto de gestión de películas, es una representación de la resolución de problemas y la aplicación práctica de conceptos de programación. Este viaje ha sido una oportunidad para aprender, crecer y, lo más importante, crear algo útil y divertido. Esperamos que disfruten usando AguMovie tanto como nosotros disfrutamos creándolo.
